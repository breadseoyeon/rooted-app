📖 프로젝트 소개
**루트 (Rooted)**는 단순히 식물을 키우는 도구를 넘어, 반려동물과 식물이 안전하게 공존하는 라이프스타일을 제안합니다. 공공 데이터(농사로, AI-Hub)와 글로벌 API(Perenual)의 정밀 데이터를 기반으로, 초보 식집사도 안심하고 '이지 플랜터'로 성장할 수 있는 환경을 제공합니다.
